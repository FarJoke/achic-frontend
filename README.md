# Achic — социальная платформа достижений
## 1. Обзор приложения
Achic — социальное веб-приложение в духе игровых ачивок (Steam, Xbox, PlayStation, внутриигровые достижения), которое позволяет пользователям:
  - создавать собственные достижения (ачивки)
  - выполнять свои и чужие ачивки
  - отмечать выполнение
  - делиться прогрессом с друзьями и видеть их активность
Цель приложения — геймифицировать повседневную жизнь и социальное взаимодействие.
Проект разрабатывается как pet‑project с фокусом на развитие навыков и знаний фронтенд разработки.
____
## 2. Стек разработки
Frontend:
  - React
  - TypeScript
  - Tailwind CSS
  - Zustand
  
Backend:
  - Node.js
  - Express
  - TypeScript
  - MongoDB 

CI/CD через GitHub → Render
____
## 3. Функциональные требования (MVP)
- Регистрация пользователя
- Авторизация (JWT)
- Просмотр профиля
- Создание ачивки
- Просмотр списка созданных ачивок
- Отметка выполнения
- Добавление друзей
- Просмотр выполненных ачивок друзей
- Лента активности (feed)
